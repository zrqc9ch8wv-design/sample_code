"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --window-position=1920,0 --window-size=1920,1080 --kiosk "C:\Users\YourName\Documents\slideshow_complete.html"
```

**修正箇所:**
- `--window-position=1920,0` → あなたのモニタ2の座標
- `C:\Users\YourName\Documents\slideshow_complete.html` → 実際のファイルパス

3. `次へ` をクリック

4. ショートカット名を入力（例: `スライドショー自動起動`）

5. `完了` をクリック

### **ステップ4: 動作テスト**

1. 作成したショートカットをダブルクリック
2. モニタ2にEdgeがフルスクリーンで開くか確認
3. スライドショーが自動再生されるか確認

**問題がある場合:**
- モニタ1で開く → 座標が間違っている
- フルスクリーンにならない → `--kiosk` が正しく入力されているか確認
- ファイルが開かない → パスが間違っている
```
